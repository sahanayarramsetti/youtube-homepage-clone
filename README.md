# 📺 YouTube Homepage Clone

This project is a **YouTube Homepage UI Clone** built using **HTML and CSS**. It replicates the layout and design of YouTube's home interface, focusing on front-end structure and styling.

---

## 🚀 Features

* Responsive YouTube-style homepage layout
* Header with logo, search bar, and icons
* Sidebar navigation menu
* Video grid layout with thumbnails
* Clean and structured UI design
* Hover effects and styling

---

## 🛠️ Technologies Used

* **HTML5** – for structure
* **CSS3** – for styling and layout
* Flexbox & Grid – for responsive design

---

## 🎯 Purpose of the Project

The main goal of this project is to:

* Practice HTML & CSS fundamentals
* Understand layout design like real-world websites
* Improve UI development skills
* Learn how to structure a front-end project

---


## ▶️ How to Run the Project

1. Download or clone the repository
2. Open `youtube.html` in your browser
